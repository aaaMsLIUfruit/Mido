# Mido
AI-powered conversation workspace where all chats are saved and managed like files, enabling seamless integration with notes, plans, and intelligent suggestions. Features also include a markdown-based note system built on a tree-structured content organization, along with AI-assisted writing and summarization.  
一个以 AI 对话为核心的生产力工作区，所有对话都可以像文件一样保存和管理，实现与笔记、计划及智能建议的无缝融合。项目还提供基于树状结构的 Markdown 笔记系统，并支持 AI 辅助写作与总结。
